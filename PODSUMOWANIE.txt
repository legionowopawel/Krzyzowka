CO TO JEST?
================================================================================

Generator Krzyżówek to subprogram dedykowany do automatycznego tworzenia  
klasycznych krzyżówek (jak w gazetach) na bazie słów z pliku tekstowego.

Funkcjonuje niezależnie jako moduł w projekcie Scrabble i tworzy:
  - Krzyżówki o wymiarach 5×5 do 30×30 komórek
  - 1-10 wariantów na jedno żądanie
  - Multi-format output (PNG, Excel, HTML, TXT)
  - Pytania numerowane (poziomo/pionowo)

================================================================================
INSTRUKCJA SZYBKIEGO STARTU
================================================================================

INSTALACJA (jednorazowo):
  C:\Krzyzowka> pip install -r requirements_krzyzowka.txt

TRYB GRAFICZNY (GUI):
  C:\Krzyzowka> python main.py
  → Interfejs pojawia się
  → Wstaw wymiary, ilość wariantów
  → Kliknij "Generuj krzyżówkę"
  → Wyniki pojawią się w folderze WYNIKI_*/

TRYB LINII POLECEŃ (CLI):
  C:\Krzyzowka> python main.py --cli 15 15
  → Generuje 15×15, 3 warianty
  → Outputs: wyniki/WYNIKI_DATE_TIME_*/

DOSTOSOWANIE BAZOWEGO PLIKU SŁÓW:
  1. Edytuj baza.txt
  2. Format: SŁOWO DEFINICJA (każde słowo w nowej linii)
  3. Zapisz (UTF-8)
  4. Uruchom generator — będzie używać nowe słowa

================================================================================
STRUKTURA WYJŚCIA
================================================================================

Każde uruchomienie tworzy folder: WYNIKI_DATA_GODZINA_PLIK/

Zawiera dla każdego wariantu (1, 2, 3, ...):

  NNN_completed.png   — Krzyżówka z wyrazami (gotowa do czytania)
  NNN_blank.png       — Pusta siatka (do druku i uzupełniania)
  NNN.txt             — Tekst z pytaniami (poziomo/pionowo)

Plus dla wariantu 1 (skonsolidowane):
  krizowka.xlsx       — Excel ze wszystkimi wyrazami
  krizowka.html       —HTML do druku lub przeglądania

RAZEM: 11 plików na jedną generację (3 warianty default)

================================================================================
PARAMETRY GENEROWANIA
================================================================================

GUI — Kontrolki:
  ▢ Szerokość siatki     : 5-30 (default 15)
  ▢ Wysokość siatki      : 5-30 (default 15)
  ▢ Liczba wariantów     : 1-10 (default 3)
  ▢ Plik ze słowami      : browse (default baza.txt)
  
CLI — Parametry:
  python main.py --cli WIDTH HEIGHT [PLIK]
  
  WIDTH              : 5-30 (szerokość)
  HEIGHT             : 5-30 (wysokość)
  PLIK (opcjonalny)  : ścieżka do pliku słów (default baza.txt)
  
Przykłady:
  python main.py --cli 10 10              # 10×10 z baza.txt
  python main.py --cli 15 20 mojeSlowa.txt # 15×20 z custom file
  python main.py --cli 8 8                # 8×8 szybko

================================================================================
FORMAT PLIKU ZE SŁOWAMI
================================================================================

Struktura: baza.txt (lub custom filename)
Format: Jedno słowo na linię, utf-8 encoded

Linia: SŁOWO DEFINICJA_LUB_PODPOWIEDŹ

Wymagania:
  ✓ WIELKIE LITERY (SŁOWA)
  ✓ Polska ortografia (obsługuje Ą,Ć,Ę,Ł,Ń,Ó,Ś,Ź,Ż)
  ✓ UTF-8 encoding
  ✓ Min. 50 słów (mniej = trudne generowanie)
  ✓ Bez pustych linii lub symboli specjalnych

Przykład (baza.txt):
  PRACA Zarobkowe zajęcie
  ZAWÓD Wyuczona profesja
  ETAT Stałe zatrudnienie na pełny etat
  NAUKA Zdobywanie nowej wiedzy
  LEKCJA Zajęcia szkolne z jednego przedmiotu
  MALARZ Osoba malująca obrazy lub domy
  DREWNO Materiał z drzewa

================================================================================
METRYKI WYDAJNOŚCI
================================================================================

Czas generowania (przetestowany):
  10×10  : ~1-2 sekundy
  12×12  : ~2-3 sekundy
  15×15  : ~3-5 sekund
  20×20  : ~10-15 sekund
  25×25  : 30-60 sekund

Gęstość wyrazów (średnia):
  10×10  : 50-55%
  12×12  : 47-51%
  15×15  : 45-55%
  20×20  : 40-50%

Liczba słów (średnia):
  10×10  : 12-17 słów
  12×12  : 17-26 słów
  15×15  : 20-30 słów
  20×20  : 25-40 słów

================================================================================
CECHY ALGORYTMU
================================================================================

✓ Backtracking z optymalizacją  — Maksymalizuje gęstość wyrazów
✓ Walidacja przecinania się      — Wyrazy muszą się przecinać (jak w gazecie)
✓ Multiwariantowość            — Każde uruchomienie daje kilka opcji
✓ Przymusowe szerokości         — Do maksymalnego wymiaru 30×30
✓ Inteligentne sortowanie       — Priorytet do wyrazów o największej liczbie przecinań

Algorytm NIE robić:
  ✗ Nie łączy się z Scrabble scoring (odrębna funkcja)
  ✗ Nie sprawdza poprawności słów w innych słownikach
  ✗ Nie generuje losowych pozycji (tylko backtracking)

================================================================================
ROZWIĄZYWANIE PROBLEMÓW
================================================================================

Problem: "ModuleNotFoundError: No module named 'PySide6'"
  Rozwiązanie: pip install -r requirements_krzyzowka.txt

Problem: "FileNotFoundError: baza.txt not found"
  Rozwiązanie: Sprawdź czy plik jest w katalogu Krzyzowka/

Problem: GUI się nie uruchamia
  Rozwiązanie: Zainstaluj PySide6 (pip install PySide6)

Problem: "UnicodeDecodeError" w baza.txt
  Rozwiązanie: Otwórz baza.txt w Notepad++, ustaw UTF-8, zapisz

Problem: Generowanie trwa bardzo długo
  Rozwiązanie: Użyj mniejszych wymiarów (10×10 zamiast 25×25)

Problem: Krzyżówka ma mało słów
  Rozwiązanie: Zwiększ bazę słów (dodaj więcej linii do baza.txt)

Problem: Python "SyntaxError"
  Rozwiązanie: Sprawdź czy Python 3.10+ (python --version)

PRZYKŁAD UŻYCIA
================================================================================

Scenariusz: Chcę wygenerować 3 krzyżówki 13×13 z moimi słowami

Kroki:
  1. Tworzę plik "moje_slowa.txt":
     
     KOMPUTER Urządzenie do obliczeń
     MONITOR Ekran do wyświetlania obrazu
     KLAWIATURA Urządzenie wejściowe z klawiszami
     MYSZKA Urządzenie sterujące kursorem
     (... więcej słów ...)

  2. Umieszczam plik w katalogu Krzyzowka/

  3. Uruchamiam:
     python main.py --cli 13 13 moje_slowa.txt

  4. Czekam 5-10 sekund

  5. Sprawdzam folder WYNIKI_20250522_120000_moje_slowa/
     
     Zawiera:
       │
       ├─ 001_completed.png  (gotowa krzyżówka)
       ├─ 001_blank.png      (pusta do druku)
       ├─ 001.txt            (pytania)
       │
       ├─ 002_completed.png
       ├─ 002_blank.png
       ├─ 002.txt
       │
       ├─ 003_completed.png
       ├─ 003_blank.png
       ├─ 003.txt
       │
       ├─ krizowka.xlsx      (skonsolidowany Excel)
       └─ krizowka.html      (skonsolidowany HTML)

  6. Drukuję PNG lub otwieraam HTML w przeglądarce
